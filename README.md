# titanic-survival-ml-flask
Machine Learning web app to predict Titanic passenger survival using Logistic Regression and Flask

This project is a complete end-to-end machine learning application built using the Titanic dataset. It predicts whether a passenger would survive or not based on input features such as age, gender, class, and fare.

The model is built using Logistic Regression and deployed with Flask as a web application. The interface allows users to input passenger details and get real-time predictions with a visually appealing UI.
